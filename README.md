```bash
      <example> - пример без знаков <>
    
Например:
      mysql -u <mysql_user> -p -d <mysql_database> < <example>.sql
выглядит как:
      mysql -u root -p -d inno_DB < inno.sql
```
```python
если пример в шаблоне будет в знаках <> , то экранирую $....$
Например:
      $<example>$
```
